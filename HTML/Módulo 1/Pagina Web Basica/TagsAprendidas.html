Estrutura

(Voltar)

Todo documento HTML deve ser estruturado com algumas tags básicas, sendo elas:

  1. !DOCTYPE html
  2. html lang="pt-BR"
  3. head
  4. meta metadados
  5. title
  6. body

Sendo o body a parte que estará visível ao usuário.

Tags

(Voltar)

Todo documento HTML será construído por tags que podem definir:

  • Títulos
  • Subtítulos
  • Parágrafos
  • Negrito
  • Sublinhado
  • Grifado
  • Listas Ordenadas
  • Listas de Tópicos

Âncoras

(Voltar)

Alguns documentos HTML poderá conter âncoras, que podem ser links que vão te direcionar para outro site ou então que te levará a outra parte do site atual, podendo ser usada para fazer botões de "Voltar ao Topo"
